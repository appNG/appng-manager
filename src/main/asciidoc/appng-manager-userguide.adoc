== General
The appNG Manager is an appNG application to configure and maintain an appNG installation and the installed applications. The manager is a mandatory core application of appNG. After installation of appNG, a site _manager_ is available with an installed manager application. The manager is available on localhost at

http://localhost:8080/manager/manager/appng-manager.


The application has different pages for the different domains of configuration and monitoring:

[cols="30,70",width="100%",options="header"]
|====================
| Page | Description
| Sites | Manage sites and application within a site
| Applications | Remove applications and edit application resources
| Users | Create users and groups and assign roles
| Repositories | Add repositories and install packages
| Platform | Configure platform properties and manage database connections.
| System | Monitor system-, environment and cluster setup. View logfile and edit logging properties.
|====================
